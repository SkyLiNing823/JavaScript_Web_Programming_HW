心理測驗--森林之旅

  1. 你一個人外出旅行，走啊走，碰到叉路，一條走向森林、另一條前往草原，你會選擇哪一條？
    2. 草原 森林

  2. 當你走到這片森林或草原時，覺得有些累了，坐下來休息，這時一隻動物經過你眼前， 你希望是什麼？
    3. 溫和型的動物(如兔子等) 溫和又笨重(如:大象等) 橫衝直撞型(如:馬、鹿等)) 兇猛動物型(如獅子、老虎等) 鳥類 小型動物

  3. 休息夠了，起身往前走，看到前方有一條河，你怎麼過？
    4. 找船或橋 涉水 游泳

  4. 過了河，前方出現一棟房子，你覺得它是怎樣的建築？
    5. 小木屋 磚瓦屋 高樓大廈 別墅 糖果屋

  5. 因為天黑了，你入內休息，你希望這個屋子有幾盞燈？
    6. 一盞燈 二盞燈 三到五盞燈 滿室燈

  6. 第二天早上你離開了這裡，希望找到一些食物，來到一間小木屋前，敲門， 你認為裡面走出來的應是怎樣的老人？
    7. 慈眉善目者 窮苦的老人 疾病纏身者 脾氣古怪者
